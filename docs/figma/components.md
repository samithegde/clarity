# Figma — Components

## Create a component

1. Select one or more frames or groups on the canvas
2. Right-click → **Create component** (or press **Ctrl+Alt+K** on Windows / **Cmd+Option+K** on Mac)
3. The frame turns purple and a diamond icon appears — this is now a Main Component

## Create a component set (variants)

1. Select multiple related components (e.g., Button/Default, Button/Hover, Button/Disabled)
2. Right-click → **Combine as variants**
3. A dashed purple border groups them into a component set
4. Use the **Variant** property panel on the right to add or rename variants

## Create an instance (use a component)

- Drag the component from the **Assets** panel onto the canvas
- Or copy-paste an existing instance

## Detach an instance

- Right-click the instance → **Detach instance** (or **Ctrl+Alt+B** / **Cmd+Option+B**)
- The instance becomes a regular group — changes no longer sync to the main component

## Reset instance overrides

- Right-click the instance → **Reset all overrides**

## Edit the main component

- Double-click the instance → click **Go to main component** in the top bar
- Or, in the Assets panel, right-click the component → **Edit main component**

## Publish components to a team library

1. Click the team name in the left sidebar → **Publish styles and components**
2. Review the changes and click **Publish**

## Use components from a library

1. Open the **Assets** panel (grid icon in the left sidebar)
2. Click the **Team library** tab
3. Enable the library you want to use
4. Drag components onto the canvas from the Assets panel

## Auto layout

1. Select a frame or component
2. Press **Shift+A** (or right-click → **Add auto layout**)
3. Use the right panel to set direction, spacing, padding, and alignment
