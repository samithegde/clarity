# Google Docs — Tables

## Insert a table

1. Click **Insert** in the top menu bar
2. Hover over **Table** in the dropdown
3. Move your cursor over the grid to choose dimensions, then click — or click **Insert table** for a custom size dialog
4. The table appears at your cursor position

## Add or remove rows and columns

- Right-click any cell → **Insert row above / Insert row below / Insert column left / Insert column right**
- Right-click a row or column header → **Delete row / Delete column**

## Resize columns

- Hover over the border between two column headers until the resize cursor appears
- Click and drag left or right to adjust width

## Merge cells

1. Select the cells you want to merge by clicking and dragging
2. Right-click the selection → **Merge cells**

## Split merged cells

- Right-click the merged cell → **Unmerge cells**

## Format table borders and background

1. Select the cells to format
2. Click **Format** in the top menu → **Table** → **Table properties**
3. Adjust border color, border width, and cell background color

## Delete a table

- Click anywhere inside the table
- Click **Format** → **Table** → **Delete table**

## Keyboard shortcuts

- **Tab** — move to the next cell (adds a new row at the end of the table)
- **Shift+Tab** — move to the previous cell
- **Tab** in the last cell of the last row — inserts a new row automatically
