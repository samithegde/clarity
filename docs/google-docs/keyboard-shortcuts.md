# Google Docs — Keyboard Shortcuts

## Text formatting

| Shortcut | Action |
|----------|--------|
| Ctrl+B | Bold |
| Ctrl+I | Italic |
| Ctrl+U | Underline |
| Ctrl+Shift+X | Strikethrough |
| Ctrl+\ | Clear formatting |
| Ctrl+Alt+0 | Normal text style |
| Ctrl+Alt+1 | Heading 1 |
| Ctrl+Alt+2 | Heading 2 |
| Ctrl+Alt+3 | Heading 3 |

## Paragraph alignment

| Shortcut | Action |
|----------|--------|
| Ctrl+L | Align left |
| Ctrl+E | Align center |
| Ctrl+R | Align right |
| Ctrl+J | Justify |

## Lists

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+8 | Bulleted list |
| Ctrl+Shift+7 | Numbered list |
| Tab | Increase list indent |
| Shift+Tab | Decrease list indent |

## Editing

| Shortcut | Action |
|----------|--------|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+C | Copy |
| Ctrl+X | Cut |
| Ctrl+V | Paste |
| Ctrl+Shift+V | Paste without formatting |
| Ctrl+A | Select all |
| Ctrl+F | Find |
| Ctrl+H | Find and replace |

## Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl+Home | Go to beginning of document |
| Ctrl+End | Go to end of document |
| Ctrl+G | Go to a specific page |
| Ctrl+K | Insert or edit link |

## Comments and suggestions

| Shortcut | Action |
|----------|--------|
| Ctrl+Alt+M | Insert comment |
| Ctrl+Enter | Submit comment |
| Esc | Cancel comment |

## Document

| Shortcut | Action |
|----------|--------|
| Ctrl+P | Print |
| Ctrl+S | Save (auto-saved, but triggers manual save) |
| Ctrl+Shift+S | Save as (make a copy) |
